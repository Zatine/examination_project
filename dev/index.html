Grid System

Set width

This is a flexible grid system built upon Flexbox. Simply add the class row to a containing div, and then give its children the class col followed by how many columns it should take up in width. You can use any number between one and twelve, where twelve fills the entire row. If the total number of columns is higher than twelve, the columns will automatically be contained to that row rather than keep their intended width.

<div class="row"> <div class="one col">One</div> <div class="two col">Two</div> <div class="three col">Three</div> <div class="six col">Six</div> </div>
<div class="row"> <div class="five col">Five</div> <div class="seven col">Seven</div> </div>
<div class="row"> <div class="twelve col">Twelve</div> </div>
